Руководство пользователя:

1) Чтобы начать ввод данных, нажмите "Clear" для очистки поля с инструкцией;

2) Ввод данных:
	2.1) Поочередно через пробел вводите необходимые данные.

	2.2) Числа должны быть в следующем порядке:
		1) coef - значение коэффициента функции;
		2) x_min - нижняя граница ограничения;
		3) x_max - верхняя граница ограничения;
		4) alpha - параметр шага градиентного спуска;
		5) max_iter - максимальное кол-во итераций
		6) х0 - начальное значение аргумента функции;

	2.3) Нажмите "Start";

3) Чтобы вернуться на предыдущую страницу, нажмите "Back".

Пример вводимых данных:
1 -2 2 0.1 3 3
